# Solving-Happy-House-Problem-with-Artificial-Neural-Network
This project develops an Artificial Neural Network (ANN) to solve a smile detection problem, where the goal is to classify whether a person in an image is smiling or not. The model is trained on a dataset of 64x64 colored face images with pixel values ranging from 0 to 255, and is built to achieve a minimum accuracy of 80% on the test set. The project covers the full machine learning pipeline including dataset visualization, exploratory analysis, model building, training, and evaluation.
